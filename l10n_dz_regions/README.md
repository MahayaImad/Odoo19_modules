# Algeria Regions Module (l10n_dz_regions)

## Description

This module provides comprehensive management of Algerian administrative regions for Odoo 17.

## Features

- **58 Wilayas**: All Algerian provinces with French and **Arabic names**
- **Communes**: Municipalities management
- **Localités**: Localities management
- **Automatic Postal Codes**: Auto-fill based on selected locality
- **Integration**: Seamless integration with:
  - Partners/Contacts
  - Companies
  - Banks

## New in This Version

- Added `name_ar` field to store wilaya names in Arabic
- Updated views to display Arabic names alongside French names
- Improved localization support for Arabic-speaking users

## Installation

1. Copy this module to your Odoo addons directory
2. Update the apps list
3. Install "Algeria - Regions (58 Wilayas)"

## Configuration

No configuration needed. The module loads all Algerian administrative data automatically.

## Usage

When creating or editing a partner, company, or bank:
1. Select **Algeria** as the country
2. Choose the **Wilaya** (province)
3. Select the **Commune** (municipality)
4. Pick the **Localité** (locality)
5. The postal code will be automatically filled
